# To Do
This is simple todo app written in sinatra, using tailwind css framework in front-end


## To install
- ``clone respository``
- ``Bundle install``
- ``rake db:create``
- ``rake: db:migrate``
- run ``rackup config.ru`` or ``bundle exec rackup -p6969 --host 0.0.0.0``
